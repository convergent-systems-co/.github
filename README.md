# .github

This repository holds **org-default community health files** for
[convergent-systems-co](https://github.com/convergent-systems-co). GitHub
applies these as fallbacks to every repo in the org that does not provide
its own.

## What's here

| Path | Purpose |
|---|---|
| `profile/README.md` | The org's profile page (rendered at github.com/convergent-systems-co) |
| `.github/FUNDING.yml` | Org-default sponsor links |
| `.github/ISSUE_TEMPLATE/` | Org-default issue templates |
| `.github/PULL_REQUEST_TEMPLATE.md` | Org-default PR template |

## Relationship to `repo-standards`

Different concerns:

- **`repo-standards`** owns the **label catalog** and the **reusable
  workflows** (triage, cleanup, sync, install).
- **`.github` (this repo)** owns the **content GitHub applies as
  org-wide defaults** when a per-repo equivalent is absent.
